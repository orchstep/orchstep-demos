# task-menu

The demo from the OrchStep blog post
[Stop memorizing commands: the task menu](https://orchstep.dev/blog/stop-memorizing-commands-the-task-menu).

```bash
orchstep menu          # fuzzy task picker with single-key hotkeys
orchstep run build     # or run a task directly by name
```

The steps only `echo`, so it is safe to run anywhere. Each task's `desc:` is what
shows up in the menu — write good ones.
